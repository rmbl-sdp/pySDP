# User guides

Longer-form walkthroughs — the Python ports of the four [rSDP vignettes](https://github.com/rmbl-sdp/rSDP/tree/main/vignettes), running against the real RMBL Spatial Data Platform.

- **[Accessing cloud data](cloud-data.md)** — catalog discovery, `open_raster` / `open_stack`, time-series slicing, when to download locally. Foundational; start here.
- **[Wrangling raster data](wrangle-rasters.md)** — cropping, masking, reprojection, resampling, combining rasters, export. The bread-and-butter GIS operations in Python.
- **[Field-site sampling](field-sampling.md)** — point extraction, buffered extraction, polygon summaries, categorical rasters, performance strategies.
- **[Pretty maps](pretty-maps.md)** — static matplotlib, interactive folium, faceted multi-panel figures, export.

Each guide is compact (1–2 screens) and mirrors the structure of its R counterpart. For the precise API surface, see the [API reference](../api.md); for an at-a-glance orientation, [Getting started](../getting-started.md).
